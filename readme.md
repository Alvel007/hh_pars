# hh_pars - Простой парсер новых вакансий на HH

## Описание 

«hh_pars» — простой парсер вакансий на всем известном сайте поиска работы. Проверяет заданную страницу на наличие обновлений и, если находит что то новое, отправляет это в чат телеги со ссылой на найденное. На сайте все параметры поиска прописываются в URL страницы, так что для корректного указания нужной вам страницы, сперва ручками перейдите в нужные параметры поиска, проставьте чекбоксы/настройки и далее копируйте строку из браузера прямо в код. Так как HH "видит" использование селениума пользователем, авторизоваться нужно на сайте непосредственно ручками (каждый раз при старте скрипта). Подставка токена не хочет работать корретно...

Стек применяемый технологий:
* Python
* Selenium

В планах (если будет скучно):
* Спрятать от сервера факт ипользования селениума.
